Warmup

Describing a Comic Book Page

Step 1: Visual Analysis

Make a mental note of the following questions:

  1. How are the panels arranged?
  2. Is there anything different about the gutter?
  3. Are there any captions, sound effects, speech or thought ballons in the panels?
  4. What are the colors like?
  5. What is going on in the foreground and background of each panel?

Step 2: Description

In pairs, take turns presenting your comic book page.

Afterwards, talk about why you think the creators have chosen some of these elements.

Group work: Analysis

Handout

  1. Start by describing the comic itself. Copy this Google document and fill out the first table.
  2. Now describe its contents, using the second table.
  3. For this particular comic, which concept lends itself to a juicy interpretation?
  4. How have the creators used historical sources to structure the comic (rhetorically and visually)? What is the effect of this?
